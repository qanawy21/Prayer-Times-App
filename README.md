# Prayer Times App – React JS

Prayer Times App is a modern, clean, and responsive web application designed to display accurate daily prayer times based on the user's selected city. Built using **React JS (Vite)** and styled with a beautiful Glassmorphism effect, this application fetches real-time data from the **Aladhan API** to provide precise timings. It is an excellent demonstration of API integration, state management, and modern UI/UX design.

## 💻 Used Technologies

* **React JS (Vite)** (Fast and optimized frontend framework)
* **CSS3** (Custom styling with modern Glassmorphism & backdrop-filter effects)
* **Aladhan API** (RESTful API for fetching accurate prayer times)
* **JavaScript** (React Hooks: `useState`, `useEffect` for state and side effects)
* **GitHub Pages** (Hosting and deployment)

---

## ✨ Key Features

* **Real-Time Data Accuracy:** Instant updates for prayer times by integrating dynamically with the Aladhan API based on the current date.
* **City Customization:** Users can select their city from a dropdown menu to view specific local prayer timings.
* **Time Formatting:** Custom logic to convert 24-hour API time formats into a user-friendly 12-hour (AM/PM) format.
* **Modern UI:** Engaging interface design featuring blur and transparency effects for a premium look.
* **Fully Responsive:** Perfectly optimized layout for all devices (Desktops, Tablets, and Mobile phones).

## 📥 Installation Instructions for Local Setup

1. **Clone the repository:**

```bash
git clone https://github.com/qanawy21/Prayer-Times-App.git

```

2. **Navigate to the project directory:**

```bash
cd Prayer-Times-App

```

3. **Install dependencies:**

```bash
npm install

```

4. **Run the development server:**

```bash
npm run dev

```
